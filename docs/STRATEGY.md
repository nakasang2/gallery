# STRATEGY.md — 競合分析と販売戦略

> ステータス: **提案**(2026-07-14 調査実施)。REQUIREMENTS.md §11(ビジネスモデル)を前提に、
> 競合調査から導いた「売れるようにするための」戦略と、現行方針への修正提案をまとめる。
> 数値・料金は調査時点の公開情報ベース(為替・改定で変動あり)。

## 1. 結論(サマリー)

1. **買い切りモデルは正しい差別化であり、変更不要。** 調査した直接競合は国内外ともほぼ全社が月額サブスク
   (Kunstmatrix €25/月〜、Exhibbit $100/月、GALLERIST3D ¥2,970/月〜)。「買い切り・無料で公開まで完結」を
   打ち出しているプレイヤーは見当たらない。ここが最大の武器。
2. **ただし、その武器がLPで一切語られていない。** 現行LPの料金セクションは旧コンセプト
   (Pro ¥980/**月**・無制限・カスタムドメイン)のままで、§11の確定方針と正面から矛盾している。最優先で修正。
3. **売上ゼロの唯一の原因は Stripe 未接続。** 課金UI・台帳・エンタイトルメントの下地(v0.43〜v0.50)は完成済みで、
   全購入ボタンが「決済はまだ準備中です」で止まっている。戦略以前の実装ボトルネック。
4. **勝ち筋は「プロダクト自体が広告になる」SNS成長ループ。** 競合(Artsteps)の唯一の堀は内蔵オーディエンス。
   HAKONIWAはSNS(X/Instagram)のクリエイターをターゲットにしている以上、
   「シェアしたくなる公開ページ」への投資が最も効率のよい集客投資になる。
5. **機能ギャップで唯一戦略級なのは「グループ展(合同展・卒展)」。** 要件定義のターゲットに「美大生/卒展主催者」が
   いるのに、対応する機能軸(複数作家で1つの箱庭)が存在しない。GALLERIST3DはMULTIプラン(¥5,500/月)を
   売っており需要は実証済み。買い切り②軸に自然に載る。

### 2.3 間接競合(クリエイターの時間と作品の置き場を奪い合う相手)

- **Instagram / X** — 現状の主戦場。課題(流れて消える・グリッドで世界観が出ない・AI学習への不信)が
  HAKONIWAの存在理由そのもの。敵ではなく**流入元**として設計する(SNSリンク・OGP・シェア導線は実装済み)。
- **Cara** — 反AIを掲げた作家向けSNS。1週間で40万→65万人、現在100万人超。
  **示唆①**: 「作品をAI学習に使わない場所」への需要は爆発的。**示唆②**: 無料+急成長でホスティング費が
  週$100kに達し運営が悲鳴 → §11.3のegress対策(R2/CDN)は正しい備え。**示唆③**: 3D展示機能はない。
- **foriio** — 国内23万人のポートフォリオサービス。無料・2Dグリッド。
  「仕事を受けるための整理」がforiio、「作品を見せる体験」がHAKONIWA、と住み分けを明確に語れる。
- **pixiv / Behance / Adobe Portfolio** — 2D一覧。展覧会体験はない。

### 2.4 市場シグナル

- 3Dアートギャラリーソフト市場はCAGR 20%超の成長予測が出ている(調査会社発表ベース)。
- **重量級メタバースは死んでいる**: Mozilla Hubs終了(2024)、NTT DOOR終了(2025)。
  ブラウザだけ・軽量・ニッチ特化(アート展示)という現在の設計思想が生存戦略として正しい。
  フェーズ2の「同時接続マルチプレイ」は優先度を上げないこと。
- コロナ以降、文化庁のデジタル化推進事業等で美術館・文化施設のオンライン展示が一般化し、
  「バーチャル展示」への心理的抵抗は下がっている。

## 3. HAKONIWAのポジショニング

```
                 高価格・業務用
                      ↑
     Exhibbit ($100/月)│ Kunstmatrix (€25〜100/月)
     GALLERIST3D(¥2,970〜)│ ArtPlacer
                      │
  画廊・機関向け ←────┼────→ 個人クリエイター向け
                      │
     Roomiq(法人)      │  Artsteps(無料・古い)
                      │  ★HAKONIWA(無料で公開+買い切り・高品質・スマホ対応)
                      ↓  oncyber(無料・Web3限定) / VRChat(無料・高摩擦)
                 無料・セルフサーブ
```

**空いているポジション = 「個人クリエイター × 無料で公開まで完結 × 買い切り × 現代的な質感 × スマホで動く × 日本語圏」。**
ここに競合が実質不在。Artstepsだけが近いが、ビジュアル品質・モバイル体験・日本語対応で明確に上回れる。

### 差別化の3本柱(顧客に語る言葉)

1. **「サブスクじゃない」** — 展示室もテーマも買い切り。やめても作品は人質にならない、公開も止まらない。
   競合全社が月額である以上、これが最強のメッセージ(唯一の例外=動画は§11の設計どおり誠実に説明)。
2. **「URLひとつで、歩ける個展」** — アプリ不要・スマホのブラウザで開ける。VRChat個展文化への回答でもある
   (「VRを持っていないフォロワーにも見せられる」)。
3. **「作品はあなたのもの」** — 販売リンクは手数料ゼロで外部EC(BOOTH等)へ直リンク(v0.49実装済み)。
   AI学習に使わない方針を規約で明文化すれば、Cara現象が証明した感情的支持を取り込める。

## 4. 販売戦略(GTM)

### 4.1 成長ループ(最重要)

競合比較で唯一負けている資産は「オーディエンス」(Artstepsの公開カタログ、pixiv/Caraのコミュニティ)。
ゼロから広告で買うのではなく、**来場体験そのものを配布物にする**:

```
作家が箱庭を公開 → SNSでURLを共有(作家の既存フォロワーが原資)
  → 来場者がスマホで歩く(=競合にない体験の驚き)
  → HUDの「Start free」(v0.49実装済み)から来場者が作家に転換
  → 新しい作家がまたSNSで共有 …
```

ループを強くする打ち手(優先順):

1. **シェア素材の自動生成**: 箱庭の30秒ウォークスルー動画/GIFをワンタップ書き出し
   (X/Instagramのタイムラインで「動く個展」は静止画OGPの数倍止まる)。技術的にも既存の順路ツアー
   カメラパスを流用できる。
2. **企画展(期間限定イベント)**: 「#夏の箱庭展」のようなテーマ公募 → Explore内特集。締切が投稿の
   きっかけを作り、ハッシュタグがSNS上の面を作る。Artstepsのカタログに対する、コミュニティ側からの対抗手段。
3. **Explore/作家ページのSEO**: 公開ページはSSR済み。「オンライン個展」「バーチャルギャラリー 無料」等の
   検索語で刺さるLP/ガイド記事(Web個展の開き方)を用意する。GALLERIST3Dはコラム記事でこれをやっている。
4. **卒展・合同展の種まき**: 美大・専門学校・イラスト系コミュニティに1件旗艦事例を作る。
   1つの合同展 = 出展作家数十人が一括でオンボードされ、それぞれのSNSフォロワーへ拡散される最効率チャネル。

### 4.2 課金の優先順位

§11の3軸(①ビデオパス/②広さ/③デザイン)は原価構造と整合しており**変更不要**。順序だけ提言:

1. **Stripe接続**(§11.7のマッピング済み。webhook→`purchases`台帳→`addWorkSlots`等、受け皿は全部ある)
2. 最初に売るSKUは**②スロット+5(¥580)と②展示室追加(¥1,480)** — 「部屋が満杯」という一番刺さる瞬間の
   アップセルUIが既に完成している(v0.46/v0.47)ため、コンバージョンの検証が最速でできる
3. ③テーマ/レイアウト新作は**有料コンテンツ第1弾(2〜3点)を早期に出す** — 現状フォーエバーフリーのみで
   🔒が1つも表示されず、購入モーダル(v0.45/v0.48)が実戦投入されていない
4. ①Video Passは配信基盤(Cloudflare Stream等、§11.3-4)を載せてから

価格の妥当性: GALLERIST3D比で「**競合の1か月分(¥2,970)より安い買い切り(¥1,480)**」という比較広告的な
訴求がそのまま成立する。値上げ余地はあるが、まず現行たたき台で転換率を計測してからでよい。

### 4.3 セグメント別

| セグメント | 刺さる訴求 | 打ち手 |
|---|---|---|
| SNSイラストレーター(主戦場) | 流れて消えない常設個展/AI学習に使わない/買い切り | 成長ループ1・2、BOOTH等への購入リンク(実装済み)を訴求 |
| 写真家 | 順路・照明・空間で世界観ごと見せる | 写真向けテーマ(モノクロ・大判)を有料コンテンツ第1弾に |
| 美大生・卒展 | 物理会場ゼロ円・無期限アーカイブ | グループ展機能(§5-4)+学校向け無償/割引プログラム |
| アートファン(来場者) | ブラウザだけで入場 | Explore・企画展。来場者は全員「将来の作家候補」(v0.49の思想を維持) |

## 5. 現行戦略への修正提案(優先度順)

| # | 修正 | 理由 | 大きさ |
|---|---|---|---|
| 1 | **LPの料金セクションを§11の確定モデルに書き換える**(現状「Pro ¥980/月・無制限・カスタムドメイン」という旧コンセプトのままで、確定済みの買い切りモデルと矛盾。無料枠の記載も「10 works」で現行の5点と不一致) | 最大の差別化(買い切り)を自ら隠している。矛盾したまま課金開始すると信頼を損なう | 小(文言・構成) |
| 2 | **Stripe接続を最優先の実装項目に** | 売上ゼロの唯一の原因。受け皿(台帳・エンタイトルメント・購入UI)は完成済み | 中 |
| 3 | **シェア用ウォークスルー動画/GIF書き出し** | 成長ループの最重要ピース。順路ツアーのカメラパスを流用可能 | 中 |
| 4 | **グループ展(複数作家で1箱庭)を②軸の買い切りSKUとしてロードマップに追加** | 要件のターゲット「美大生/卒展主催者」に対応機能がない。GALLERIST3D MULTI(¥5,500/月)が需要を実証。「グループ展パック(買い切り)」は月額のMULTIに対する明確な価格破壊になる | 大(権限モデル拡張) |
| 5 | **「作品をAI学習に使わない」を規約とLPに明文化** | Cara(100万人)が実証した、このターゲット層で最も強い感情的訴求のひとつ。コストほぼゼロ | 小 |
| 6 | **企画展(期間限定・特集)の運営メカニズムをExploreに追加** | オーディエンス不在(vs Artstepsカタログ)への対抗。手動キュレーションから始めてよい | 中 |
| 7 | マルチプレイ(フェーズ2)の**優先度を下げる方針を明文化** | DOOR・Mozilla Hubsの死因。軽量・ニッチ特化が生存戦略 | 方針のみ |
| 8 | 後回しでよいもの: AR(GALLERIST3Dにあるが決め手ではない)、カスタムドメイン、本格的な英語圏展開(UIは既に英語ベースなので土台は保持) | 差別化の核ではない | — |

### 変えなくてよいもの(調査による裏付け)

- **買い切り中心の3軸課金**(§11.5) — 全競合が月額の中で唯一の買い切り。差別化として機能する
- **無料で公開まで完結・作品を人質にしない**(§11.1) — Kunstmatrixの「無料では公開不可」、
  GALLERIST3Dの「無料枠なし」の裏返しがそのまま優位性になる
- **egress対策(R2/CDN/遅延ロード)**(§11.3) — Caraの週$100k事件が「無料で伸びるサービス」の
  インフラリスクを実証。先回りは正しい
- **無料5点**(§10.10) — Kunstmatrix STARTERの10点(ただし公開不可)、ARTWORKS βの17点と比べても、
  「公開できる5点」は試用価値と課金動機のバランスとして妥当
